# AtlasAndroid

**AtlasAndroid** is a modern Android starter project built with **Kotlin 2.0** and **Jetpack Compose**, designed to demonstrate a clean project structure, up-to-date tooling, and a reliable CI pipeline.

This repository is suitable as:
- a modern Android starter template
- a learning project for Kotlin + Compose
- a portfolio-ready project showcasing CI and build automation
